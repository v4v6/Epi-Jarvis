## Product idea
A CRUCIX-like intelligence terminal for California epidemiological events, but adapted for scientific public-health surveillance:
- county-level map intelligence
- surveillance signal rails
- outbreak / advisory ticker
- multi-source concordance logic
- uncertainty-aware public health operations view
